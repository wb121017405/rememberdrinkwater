# 提醒喝水小助手☕

这只是一个计时器，每小时弹出一次提醒。

预防肾结石、颈椎病、腰椎间盘突出、痔疮、脱发、焦虑，和我一起成为一天八杯水的人吧!

## 答应我，记得多喝热水

喝水步骤：

1. 起身
2. 去接水
3. 回来坐下
4. 水喝得多所以频繁尿急需要上厕所走动
5. 步行，看看窗外

**在打开vscode之后沉迷代码只能用内部提示提醒自己喝水，多喝水的同时，也能起身多走动**

## 这也可以是一个提肛运动小助手

提肛步骤：

1. 夹紧
2. 坚持5秒
3. 放松
4. 反复循环20次

**久坐容易痔疮，常做提肛运动能够有效预防痔疮等肛周疾病，促进血液循环**
